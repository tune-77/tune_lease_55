# タム散歩ログ — iOSショートカット設定手順

## 完成イメージ
散歩後にSlackへ自動通知：
```
🐾 タム朝散歩ログ｜2026/03/24
歩行距離: 2.4 km
歩数: 3,200歩
```

---

## iPhone設定手順（所要5分）

### STEP 1: ショートカットアプリを開く
iPhoneの「ショートカット」アプリ → 右上「＋」で新規作成

### STEP 2: アクションを追加

以下の順番で「アクションを追加」してください：

**① ヘルスケアデータを取得**
- アクション検索：「ヘルスケアサンプルを検索」
- 種類：「歩行距離＋走行距離」
- 期間：「今日」
- 集計：「合計」

**② 歩数を取得**
- アクション検索：「ヘルスケアサンプルを検索」
- 種類：「歩数」
- 期間：「今日」
- 集計：「合計」

**③ テキストを作成**
- アクション：「テキスト」
- 内容（コピペ）：
```
🐾 タム散歩ログ｜[現在の日付]
歩行距離: [①の結果] km
歩数: [②の結果] 歩
```

**④ Slackに送信**
- アクション検索：「URLの内容を取得」
- URL：`YOUR_SLACK_WEBHOOK_URL`
- 方法：POST
- リクエスト本文：JSON
  - キー：`text`
  - 値：③のテキスト変数

### STEP 3: 自動化設定（時間トリガー）

「オートメーション」タブ → 「＋」→「時刻」

**朝用：**
- 時刻：6:30
- 繰り返し：月〜金
- アクション：上記ショートカットを実行

**夜用：**
- 時刻：20:00
- 繰り返し：月〜金
- アクション：上記ショートカットを実行
