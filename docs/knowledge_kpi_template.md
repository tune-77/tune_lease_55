# Knowledge KPI Template

内部運用用の下書き。UI には出さない。

## Template

```text
【知識KPI】{指標}: {現状} -> {目標} / 変化: {何が変わるか} / 根拠: {観測できる証拠} / 次回: {いつどう確認するか}
```

## Metrics

- `知識化率`: 会話や改善案を知識ノートへ昇格できた割合
- `再利用率`: 追加した知識が次回の回答や判断で使われた割合
- `効果率`: 再利用後に応答品質や判断精度が改善した割合
- `重複・陳腐化率`: 重複ノートや参照されないノートの割合

## Use

- 改善ログには要約ではなく、測定可能な変化を書く
- 月次レビューでは、作業件数よりも再利用と効果を優先して見る
- テンプレートは内部確認用にのみ使い、画面には出さない
