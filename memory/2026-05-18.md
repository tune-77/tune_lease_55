## Codex

### NEXT版入力単位の百万円化
- Next版の審査入力とリースくん入力で、金額系フィールドの表示・入力単位を千円から百万円へ変更した。
- API/scoring_core 側は従来どおり千円前提なので、`frontend/src/lib/scoringUnits.ts` の `toThousandYenPayload()` で送信直前に百万円から千円へ換算する方針にした。
- `npx tsc --noEmit` と `npm run build` は成功。`npm run lint` は既存の unrelated な `any` / `next.config.ts` 等のエラーで失敗。
