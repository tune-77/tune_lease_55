# Obsidian Wiki Workflow for tune_lease_55

## Purpose
Make project notes easy to search and easy to traverse.

## Rules

1. Keep `Projects/tune_lease_55/tune_lease_55 Wiki.md` as the hub.
2. Add or maintain `Projects/tune_lease_55/検索語インデックス.md`.
3. Put `## Related` in each major project note.
4. Add synonyms, abbreviations, English names, file names, and API names near the top of the note.
5. Keep one note focused on one topic.
6. Write a sanitized daily log when the change is meaningful.
