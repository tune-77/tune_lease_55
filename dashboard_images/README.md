# 分析ダッシュボード用画像

このフォルダに以下のファイル名で画像を置くと、分析結果のダッシュボードに表示されます。
表示は **承認レベル**・**業種**・**物件** に応じて自動で選ばれます。

## 推奨ファイル名

| ファイル名 | 使われる場面 |
|------------|----------------|
| `approved.png` | 判定が「承認圏内」のとき（業種専用画像がない場合） |
| `review.png` | 判定が「要審議」のとき |
| `construction.png` | 業種が建設業のとき |
| `nurse.png` | 業種が医療・福祉のとき |
| `vehicle.png` | 業種が運輸のとき（任意） |
| `default.png` | 上記に当てはまらないとき |

1種類だけでも構いません（例: `default.png` のみ）。  
サイズは任意ですが、縦長で 200〜400px 程度だとバランスが良いです。

## 別フォルダの画像を使う場合

環境変数 **DASHBOARD_IMAGES_ASSETS** に画像フォルダのフルパスを指定すると、そのフォルダ内の所定のファイル名（建設・医療・桜・困惑など）が参照されます。
