## 提出概要文ドラフト（フォーム用・そのまま貼れる形）

**タイトル案**: 審査担当者の判断力を育てるAI — 自ら改善を提案し、人間の承認で育つリース審査エージェント

**概要（300字級）**:
リース案件の入力から、信用スコア・リスク論点・承認条件・稟議コメント案までを一気通貫で生成する審査支援AIエージェント。ADKエージェント（軍師AI）が審査エンジン・過去案件1900件・業種ベンチマークをツールとして自律的に呼び分け、AI人格「紫苑」が過去の判断記憶を根拠付きで提示する。Cloud RunにAPI/Webを分離デプロイし、毎朝の改善パイプラインでAI自身が改善候補（REV）を起票、人間承認を経たものだけが本番昇格するDevOpsループを個人で数ヶ月運用している。判断を自動化するのではなく、人間の判断力を育てることに軸を置いた実務エージェント。

**使用技術**: Cloud Run（API/Web分離）/ Gemini API / Agent Development Kit / Next.js / FastAPI / SQLite
