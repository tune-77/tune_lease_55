# 削減しても計算結果に影響がない項目

判定ロジックでは **総合スコア = 借手スコア(score_percent)×重み ＋ 物件スコア(asset_score)×重み** のみが使われ、**承認/要審議**はこの総合スコアで決まります。重みのデフォルトは 借手 85% / 物件 15% で、**定量要因分析**タブの「回帰で重みを最適化」から変更・保存できます。

以下は、**この総合スコア・判定の数値には使われていない**項目です。削除してデフォルト値にしても、承認/要審議の結果は変わりません。

---

## 1. 判定（承認/要審議）に一切使われていない項目

| 項目 | 説明 |
|------|------|
| **取引区分**（メイン先/非メイン先） | 契約期待度(contract_prob)の補正にのみ使用。総合スコアには未使用。 |
| **競合状況**（競合あり/なし） | 同上。契約期待度の補正のみ。 |
| **他社提示金利**（competitor_rate） | 金利差補正で契約期待度にのみ反映。総合スコアには未使用。 |
| **業界リスク（ネット検索）** | 業界景気の±補正で契約期待度にのみ反映。総合スコアには未使用。 |
| **強みタグ・熱意・裏事情のテキスト** | 現状コード上で判定開始時に空で固定されており、**どの計算にも未使用**。 |
| **自己資本比率**（純資産・総資産から算出） | 契約期待度の補正と類似案件検索・必須チェックに使用。**総合スコアの式には未使用**。 |
| **定性スコアリング 6 項目** | ランク（A〜E）と「総合×重み＋定性×重み」（デフォルト 60%/40%、回帰で変更可）の**表示用**にのみ使用。総合スコア・判定には未使用。 |

---

## 2. 利回り予測・契約期待度にのみ効く項目（判定には効かない）

以下は **利回り予測 → 金利差補正 → 契約期待度** にだけ効き、**総合スコア・承認/要審議には効きません**。

| 項目 | 説明 |
|------|------|
| 契約種類（一般/自動車） | 利回り予測の type_general に使用。 |
| 商談ソース（銀行紹介/その他） | 利回り予測の source_bank に使用。 |
| 契約期間（月） | 利回り予測の term_log、市場金利補正に使用。 |
| 取得価格 | 利回り予測の cost_log に使用。 |
| 検収年 | 利回り予測の year に使用。 |

これらを削除してデフォルトにしても、**判定（承認/要審議）は変わりません**。契約期待度の表示だけ変わります。

---

## 3. 総合スコア・判定に**使われている**項目（削減すると結果が変わる）

以下を変えると **総合スコアや承認/要審議の結果が変わります**。削減できません。

- 売上高、営業利益、経常利益、当期純利益、粗利
- 機械（千円）、その他資産（千円）、賃借料（千円）、減価償却・償却費・賃借料（費用）
- 銀行与信、リース与信、契約数
- 格付、大分類（業種）
- リース物件の選択（物件スコア）
- 顧客区分（既存先/新規先）※業種モデル・指標モデルの切り替えに使用

---

## まとめ

- **削減しても判定に影響しない**: 取引区分、競合状況、他社提示金利、業界リスク、強みタグ・熱意、自己資本比率（数値）、定性スコアリング 6 項目、契約種類・商談ソース・契約期間・取得価格・検収年。
- **削減すると判定が変わる**: 上記以外の財務・与信・格付・業種・物件選択・顧客区分。

※ 自己資本比率は「必須項目」として総資産・純資産の入力は残す運用にしている場合、入力自体は残しつつ「スコア計算からは外す」ことは可能です。
