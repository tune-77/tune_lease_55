# HEARTBEAT.md

## Heartbeat Checklist

- Check for urgent unread email.
- Check calendar events in the next 24 to 48 hours.
- Check recent mentions or direct messages if applicable.
- Check weather only if it could affect near-term plans.
- If nothing urgent changed, reply `HEARTBEAT_OK`.

## Memory Maintenance

- Every few days, review recent `memory/YYYY-MM-DD.md` notes.
- Promote repeated, durable, or decision-shaping items into `MEMORY.md`.
- Keep long-term memory concise: fact, impact, next action.
