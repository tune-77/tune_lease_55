# 工作機械の残価・再販リスク

## 要点

工作機械は中古流通があるが、主軸稼働時間、制御装置の世代、メーカー、搬出・据付費で実質回収額が変わる。

## 判断ルール

- CNCやマシニングセンタは制御装置の世代を確認する。
- 主軸稼働時間が長い場合は整備履歴と精度維持を確認する。
- 大型機は搬出費、基礎工事、再据付コストを控除して見る。
- 特注機・専用機は汎用機より再販リスクを高く見る。

## 根拠

中古市場がある物件でも、実際の換価額は「売れる価格」から撤去・搬出・再整備コストを引いた金額になる。
