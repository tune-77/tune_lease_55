# リース審査：業種別ポイントと金利相場

**作成日**: 2026-05-30  
**情報源**: ネット検索  
**対象**: 業種による審査の違いと金利の相場観

---

## 🎯 業種による審査の難易度

### ✅ 審査が通りやすい業種（信頼度：高）

#### 1. **士業・専門職**
- 医師、歯科医師、獣医師
- 弁護士、公認会計士、税理士
- 社労士、行政書士
- 理由: 国家資格が必須 → 社会的信頼性が高い

**審査難度**: ⭐（最容易）  
**必要書類**: 身分証、決算書（最近1期のみでも可）

#### 2. **公務員関連**
- 国家公務員、地方公務員
- 独立行政法人職員
- 理由: 安定した雇用、給与が保証

**審査難度**: ⭐  
**金利**: 最低水準

#### 3. **大企業の関連会社・子会社**
- 親会社の経営基盤がある
- グループ全体の支援を期待できる

**審査難度**: ⭐⭐  
**必要提出**: 親会社の支援確認書があると有利

#### 4. **建設業（元請け・実績豊富）**
- 公共工事実績が豊富
- 営業許可をもっている
- 複数年の安定した受注

**審査難度**: ⭐⭐⭐  
**必要書類**: 営業許可証、入札実績書

---

### ⚠️ 審査が慎重になる業種（信頼度：中）

#### 1. **製造業**
```
装置産業（土地・建物・機械が必要）
   ↓
担保価値がある
   ↓
リース対象資産と関連性がある
   ↓
審査通りやすい
```

**審査難度**: ⭐⭐⭐  
**特徴**: 以下があると有利
- 鉄骨造以上の工場を所有
- 機械設備の定期更新がある
- 長期顧客との契約がある

#### 2. **物流業・運送業**
```
トラック・フォークリフトなどが対象
   ↓
資産が実物（担保になる）
   ↓
相場が明確（残価リスク低い）
   ↓
比較的通りやすい
```

**審査難度**: ⭐⭐⭐

#### 3. **医療機関（病院・クリニック）**
```
器具・装置が必要
   ↓
更新サイクルが明確
   ↓
保険診療で安定収入
   ↓
通りやすい
```

**審査難度**: ⭐⭐⭐

---

### ❌ 審査が厳しい業種（信頼度：低）

#### 1. **飲食業（個人経営）**

```
理由：
・短期間で閉店する率が高い
・売上が天候・景気の影響を受けやすい
・夜間営業で従業員管理が困難
・原価率が高い（利益率が低い）
```

**審査難度**: ⭐⭐⭐⭐⭐（最難）  
**対応策**:
- ✅ チェーン店（親会社支援）: 審査可能
- ✅ 実績3年以上で売上安定: 審査可能
- ❌ 開業1年以内: ほぼ不可

#### 2. **美容・エステ業**
```
理由：
・開業資金が少ない（実力不問で開業可）
・流行に左右されやすい
・オーナーの技術力に依存
・顧客の固定化が難しい
```

**審査難度**: ⭐⭐⭐⭐⭐  
**対応策**: 
- ✅ 複数店舗経営: 審査可能
- ✅ 安定した顧客基盤: 審査可能

#### 3. **風俗業**
```
理由：
・社会的な位置付けの問題
・規制が厳しい
・従業員管理が難しい
```

**審査難度**: ⭐⭐⭐⭐⭐  
**対応**: リース会社によっては対応不可の場合あり

#### 4. **個人タクシー・運送**
```
理由：
・個人経営で規模が小さい
・事故リスクが高い
・使用権資産の残価が不確定
```

**審査難度**: ⭐⭐⭐⭐  
**対応策**:
- ✅ 法人化: 審査通りやすくなる
- ✅ 複数台: 経営基盤が見える

## 💰 リース金利相場（2026年）

### 金利の決定要因

```
基準金利
   ↓
+ リース会社のマージン
   ↓
+ 業種リスクプレミアム
   ↓
+ 信用リスク調整
   ↓
= 提示される金利
```

### 標準的なリース料率

#### OA機器（複合機・プリンタなど）

```
5年リース: 1.9% ～ 2.0%
6年リース: 1.6% ～ 1.7%
3年リース: 2.5% ～ 3.0%（短期割高）
```

**計算例** （複合機：300万円、5年リース）:
```
月額料金 = 300万円 × 1.95% ÷ 60ヶ月
        = 300万円 × 0.0195 ÷ 60
        ≈ 9,750円/月
```

#### 自動車（トラック・営業車）

```
乗用車: 2.5% ～ 3.5%
トラック: 2.0% ～ 3.0%
特殊車両: 3.0% ～ 4.0%
```

#### 建設機械・重機

```
ブルドーザー、ショベルカー: 2.5% ～ 3.5%
クレーン: 3.0% ～ 4.0%
フォークリフト: 2.0% ～ 3.0%
```

## 📊 金利に影響を与える要因

### 金利が低くなる要素（有利）

```
✅ 黒字決算       → 信用度が上がる
✅ 自己資本比率高 → 経営基盤が堅い
✅ 滞納なし       → 支払い履歴が良好
✅ 長期契約       → リース会社の安心
✅ 担保あり       → デフォルトリスク低い
```

### 金利が高くなる要素（不利）

```
❌ 赤字決算       → 信用度が下がる
❌ 自己資本比率低 → 経営基盤が弱い
❌ 滞納履歴あり   → 支払い懸念あり
❌ 短期契約       → リース会社のリスク高
❌ 無担保         → デフォルトリスク高い
```

## 🌍 2026年の金利環境の特徴

### 日銀政策の変化（2024年3月以降）

```
マイナス金利: 解除
    ↓
イールドカーブコントロール: 廃止
    ↓
長短金利: 上昇傾向
    ↓
リース金利: 上昇傾向
```

### 企業への影響

```
2026年時点:
・金利全体が上昇傾向
・属性再審査が厳しくなる可能性
・信用スコアが下がると金利上乗せ増加

注意：
・2024年初と比べると金利が高い
・借入当初より属性低下の企業は審査困難
・タイミングによって金利に0.5-1.0% の開き
```

## 📋 業種別リース活用パターン

### 製造業

**リース対象**: 機械設備、フォークリフト、検査装置
**金利**: 2.0% ～ 3.0%（担保価値がある）
**審査期間**: 1-2週間

### 建設業

**リース対象**: ユンボ、バックホー、仮設足場
**金利**: 2.5% ～ 3.5%（重機の相場が明確）
**審査期間**: 1週間

### 飲食業

**リース対象**: 厨房機器、POS システム
**金利**: 3.5% ～ 5.0%（リスク調整）
**審査期間**: 2-3週間

### 物流・運送

**リース対象**: トラック、フォークリフト、配送車
**金利**: 2.0% ～ 3.0%（資産価値が明確）
**審査期間**: 1-2週間

## 💡 金利交渉のコツ

### 1. 複数のリース会社から見積もり取得
```
→ 金利を比較できる
→ 交渉材料になる
→ 1% の差でも年間数万円の違い
```

### 2. 複数ユニットのリース
```
複合機3台 → 単体より安い金利
トラック5台 → ボリュームディスカウント
```

### 3. 長期リース（有利）
```
3年: 高い金利
5年: 標準的
7年: 安い金利
（リース会社は長期を好む）
```

### 4. 担保の提供
```
不動産を担保に → 金利 -0.5%
既存設備を担保に → 金利 -0.3%
```
