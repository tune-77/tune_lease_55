# リース審査：残価リスク評価と市場動向

**作成日**: 2026-05-30  
**情報源**: ネット検索  
**対象**: リース会社が抱える残価リスク理解

---

## 📌 残価（残存価値）とは

### 定義

```
新品購入価格
   ↓ (リース利用で減価)
5年リース満了時の予想売却価格 = 残価
```

**例**:
```
複合機: 300万円で購入
5年後のリース満了時
→ 予想売却価格: 60万円（20%）
→ 残価: 60万円
```

---

## 💰 オペレーティングリースの仕組み

### リース会社のビジネスモデル

```
新品購入: 300万円（リース会社が負担）
   ↓
ユーザーから月々リース料を回収
5年 × 月9,750円 = 585万円
   ↓
リース満了時に中古品を売却
売却価格: 60万円
   ↓
リース会社の利益 = 585万円 - 300万円 - 運用コスト - 売却コスト + 60万円
```

### リース会社が設定する残価の目安

#### 複合機・OA機器
```
1年後: 80-85%
2年後: 60-70%
3年後: 40-50%
4年後: 20-30%
5年後: 10-20%
6年後: 5-10%
```

**背景**: 技術進歩が速い → 古い機器の価値が急速に低下

#### 自動車
```
1年後: 85-90%
2年後: 75-80%
3年後: 60-70%
4年後: 50-60%
5年後: 40-50%
```

**背景**: 走行距離、事故歴で価値が変動

#### 建設機械
```
1年後: 90-95%
2年後: 80-90%
3年後: 70-80%
4年後: 60-70%
5年後: 50-60%
```

**背景**: 使用環境によって寿命差が大きい

## 🚨 残価リスク：リース会社の損失パターン

### パターン 1: 技術革新による価値低下

```
予想残価: 60万円（予定）
実際の売却価格: 30万円（予想外の技術進歩）
リース会社の損失: 30万円
```

**具体例**:
- 複合機: カラー複合機の価格急騰 → 白黒機の価値暴落
- サーバー: クラウド普及 → オンプレミスサーバーの需要激減
- 製造機械: 自動化進展 → 旧型機械の需要激減

### パターン 2: 景気変動による相場下落

```
予想残価: 200万円（景気好況想定）
実際の売却価格: 120万円（景気悪化で中古市場冷え込み）
リース会社の損失: 80万円
```

**具体例**（2008年金融危機）:
- トラック: 300万円 → 150万円に暴落
- 建設機械: 1000万円 → 400万円に暴落

### パターン 3: 使用状況による劣化

```
予想残価: 100万円（標準的な使用想定）
実際の売却価格: 40万円（24時間稼働で劣化）
リース会社の損失: 60万円
```

### パターン 4: リース会社の過度な楽観的評価

```
市場相場: 50万円
リース会社の評価: 80万円（希望的観測）
実際の売却価格: 45万円
リース会社の損失: 35万円
```

## 📊 2026年の市場動向

### 自動車市場

#### EV シフトによる中古車価値の変動
```
ガソリン車:
・供給増加 → 価値下落傾向
・電池技術進展で残価不透明

EV車:
・需要拡大で価値保持
・ただし電池劣化が不確定
```

#### 特徴:
```
✅ 新型国産EVの価値は保持
❌ 旧型ガソリン車は価値下落
❌ 輸入EV の残価が不透明
```

### 製造機械・OA機器市場

#### 不確定要因
```
AI の進展:
・汎用機械の需要減少
・特殊機械の高差別化

クラウド普及:
・サーバーリース化
・オンプレミス機械の価値低下

半導体不足解消:
・機械の新型投入加速
・旧型機械の陳腐化スピード加速
```

### 建設機械市場

#### インフレと金利上昇の影響
```
2025年のインフレ: 金融引き締め
   ↓
建設投資: 抑制傾向
   ↓
中古重機需要: 低迷
   ↓
残価: 予想より低い可能性高い
```

## 🔍 リース会社が残価リスクを管理する方法

### 1. **保守的な残価設定**
```
予想売却価格: 60万円
実際の設定: 50万円（10万円の安全余裕）
→ 予想を上回る売却価格 = リース会社の利益増
→ 予想を下回る売却価格 = 損失を最小化
```

### 2. **リース料の設定で調整**
```
残価リスク高い機械:
・月額リース料を高めに設定
・リース期間を短めに設定（例: 3年）

残価リスク低い機械:
・月額リース料を低めに設定
・リース期間を長めに設定（例: 7年）
```

### 3. **下取りプログラム**
```
新型機械へのリース乗り換え
   ↓
旧型機械は下取り
   ↓
リース会社が市場価格で回収
→ 残価予想に頼らない方式
```

### 4. **有担保リース**
```
ユーザーが不動産を担保提供
   ↓
残価リスク低下 → 金利低下
   ↓
リース会社のリスク軽減
```

## 💡 ユーザーが知っておくべき点

### 1. リース料金に残価リスクが含まれている

```
複合機のリース料：月9,750円
   ↑
この中に:
・複合機の価値低下分
・リース会社の予想残価ギャップへの保険
・リース会社のマージン
が全部含まれている
```

### 2. 短期リースは割高

```
理由：
短期リース = 残価が高い（予測誤差が大きい）
   ↓
リース会社がリスク回避で金利を上げる
   ↓
ユーザーが負担
```

**例**:
```
3年リース: 月10,500円（1.95% + リスク調整）
5年リース: 月9,750円（1.95% ベース）
7年リース: 月9,200円（1.95% - リスク低減）
```

### 3. 償却資産税の逆転現象

```
通常: リース期間 = 償却期間
例外: リース期間が短い場合
   ↓
月額リース料が高い（残価リスク高いため）
   ↓
全体で支払う総額が高くなることも
```

## 📈 残価リスク対策のポイント

### ユーザーの視点

#### ✅ 残価リスク低い機械を選ぶ
```
・使用期間が確定している（償却資産と一致）
・技術変化が遅い分野
・市場需要が安定している
```

**例**:
```
✅ トラック（市場相場が明確）
✅ 一般的な複合機（市場が確立）
❌ AI 関連機械（技術が急速に進展）
❌ EV 充電器（規格が不安定）
```

#### ✅ 長期リースで金利低下
```
長期 = 残価リスク低い
   ↓
金利が下がる
   ↓
月額料金が安くなる
```

#### ✅ 実績のあるリース会社を選ぶ
```
大手リース会社：
・市場データが豊富
・残価予測が正確
・金利が有利

小規模リース会社：
・残価予測が外れやすい
・リスク調整で金利が高い
```

## 🎯 まとめ

### リース会社の視点

```
残価リスクが高い
   ↓
月額リース料が高い
   ↓
金利が高い
   ↓
または短期リースしか受け付けない
```

### ユーザーの視点

```
残価リスク低い機械を選ぶ
   ↓
金利が有利になる
   ↓
月額料金が安くなる
   ↓
または長期リースで追加割引
```

### 2026年の特徴

```
・技術進歩スピード加速 → 残価リスク上昇傾向
・景気不透明 → 残価予測が困難
・EV シフト → 自動車残価が不透明
・AI 活用 → 機械残価が不透明
   ↓
保守的な金利設定 → ユーザーの負担増加
```
