# リース審査：決算書分析と財務指標

**作成日**: 2026-05-30  
**情報源**: ネット検索  
**対象**: 決算書を使った正確な企業判定

---

## 📊 決算書の3つの構成要素

### 1. 貸借対照表（B/S: Balance Sheet）

**企業の財務状態を示す**
```
左側（資産）         右側（負債 + 資本）
現金・預金          短期借入金
売掛金              長期借入金
在庫                ─────────
機械・建物          自己資本
                   利益剰余金
```

**リース審査で見るポイント**:
- **自己資本比率** = 自己資本 ÷ 総資産
  - ✅ 30% 以上: 健全
  - ⚠️ 20-30%: 要注意
  - ❌ 20% 未満: リスク高

- **流動比率** = 流動資産 ÷ 流動負債
  - ✅ 100% 以上: 支払い能力あり
  - ❌ 100% 未満: 短期的に危険

- **負債内容**
  - リース料が支払える自由度があるか
  - 既存の借入金が多すぎないか

### 2. 損益計算書（P/L: Profit & Loss）

**その期間の収益性を示す**
```
売上高
   ↓
営業外収益（金利収入など）
   ↓
営業利益 = 売上高 - 営業費用
   ↓
営業外費用（支払い金利など）
   ↓
経常利益
   ↓
特別利益・損失（臨時的な利益・損失）
   ↓
当期利益 = 最終的な利益（損失）
```

**リース審査で見るポイント**:
- **経常利益**: リース料支払い能力を判定
  - ✅ プラス: 基本的に通る
  - ⚠️ 赤字（1期だけ）: 個別判定
  - ❌ 赤字（2期以上）: 通らない可能性高

- **売上高の推移**
  - 右肩上がり: 成長企業として評価
  - 横ばい: 安定企業として評価
  - 右肩下がり: リスク企業として判定厳しく

- **営業利益率** = 営業利益 ÷ 売上高
  - ✅ 5% 以上: 良好
  - ⚠️ 2-5%: 標準的
  - ❌ 2% 未満: 利益性が低い

### 3. キャッシュフロー計算書（CF: Cash Flow）

**実際の現金の動き（手元の現金）**
```
営業CF（事業から生まれた現金）
   ↓ ここが重要
投資CF（設備投資などで使った現金）
   ↓
財務CF（融資を受けた・返した現金）
   ↓
期末現金残高
```

**リース審査で見るポイント**:
- **営業CF**: 実際の支払い能力を示す
  - ✅ プラス: 事業で現金を生み出している
  - ❌ マイナス: 現金不足の危険信号

**重要**: 利益がプラスでも、営業CFがマイナスなら危険

## 💰 リース審査で使う重要財務指標

### 1. **売上高** （リース料支払い能力）
```
リース月額 10万円の場合
  ↓
年間リース料 = 120万円
  ↓
必要な年間売上: 1000万円以上が目安
  ↓
売上に対するリース料比率 = 120万 ÷ 1000万 = 12%（許容範囲）
```

**判定基準**:
- ✅ 売上 1000万円以上: リース対応可能
- ⚠️ 売上 500-1000万円: 規模を制限
- ❌ 売上 500万円以下: 少額リースのみ

### 2. **経常利益** （企業の純粋な収益性）

```
営業利益 = 本業の利益
   ↓
営業外費用を引く
   ↓
経常利益 = 日常的な利益（リース支払い能力判定に使用）
```

**判定基準**:
- ✅ 経常利益 > 0（プラス）: 通常通過
- ⚠️ 経常利益 < 0（赤字）: 1期だけなら相談可
- ❌ 経常利益 < 0（赤字）: 2期以上は困難

### 3. **自己資本比率** （企業の健全性）

```
自己資本比率 = 自己資本 ÷ 総資産 × 100%
```

**業種別の目安**:
| 業種 | 目安 |
|------|------|
| 製造業 | 30% 以上 |
| 建設業 | 25% 以上 |
| 卸売業 | 20% 以上 |
| 小売業 | 15% 以上 |

### 4. **流動比率** （短期的な支払い能力）

```
流動比率 = 流動資産 ÷ 流動負債 × 100%
```

**判定基準**:
- ✅ 100% 以上: 短期負債を短期資産でカバー可能
- ⚠️ 80-100%: 要注意
- ❌ 80% 未満: リスク高

### 5. **営業CF** （実際の現金生成能力）

```
営業CF = 事業活動から生まれた現金
```

**重要**: 利益はプラスでも、営業CF がマイナスなら警戒必要

## 🔴 赤字決算への対応

### 赤字が出た場合

**1期のみ赤字**:
- ✅ 対応: 相談可能
- 対策: 理由説明（一時的な事象か、構造的か）
- 例: 新工場立上げ、災害対応、景気変動

**2期以上連続赤字**:
- ❌ リース審査: 非常に困難
- 必要: 経営改善計画の提出
- 期待: 直近決算で黒字化の兆し

**累積赤字で債務超過**:
- ❌ リース審査: ほぼ不可能
- 理由: 返済能力の疑問

## 📈 3期比較分析（推奨）

リース会社は決算書を3期分見て、経営の**トレンド**を判定

```
1期目: 100万円利益
2期目: 150万円利益  ← 右肩上がり = ✅ Good
3期目: 200万円利益

vs

1期目: 200万円利益
2期目: 150万円利益  ← 右肩下がり = ⚠️ Risk
3期目: 100万円利益
```

## 📊 新リース会計基準（IFRS 16）の影響

### 2023年以降の変更

**従来**:
- 簿外リース: 貸借対照表に表示されない
- オペレーティングリースを使う企業が多かった

**新基準**:
- すべてのリース: 貸借対照表に表示
- 使用権資産として計上
- リース債務として計上

### 決算書への影響

```
資産側:   ↑ 使用権資産が増加
          ↑ 総資産が増加

負債側:   ↑ リース債務が増加
          ↑ 総負債が増加

利益:     ↑ EBITDA（営業利益）が改善
          ↓ 営業利益は悪化
          ↓ 当期利益は変わらず
```

**リース審査への影響**:
- 昔のデータ（簿外リース）と比較が困難
- 企業の見かけの負債が増える
- EBITDA を重視する傾向へシフト

## 💡 リース審査に向けた決算書の準備

### 1. 数字の信頼性
```
□ 帳簿が正確に記録されているか
□ 脱税の疑いがないか
□ 臨時的な利益・損失が多くないか
```

### 2. 決算書の整理
```
□ 3期分の決算書を用意
□ 附属書類（製造原価明細書など）も含める
□ 税務申告書との一致確認
```

### 3. 説明資料の準備
```
□ 赤字の場合: 理由説明書
□ 業績の大きな変動: 背景説明
□ 新規事業: 事業計画書
```

---

## 📌 まとめ

リース審査では決算書の以下の順で重視される：

```
1. 経常利益の黒字化
   ↓
2. 売上高の安定性と増加傾向
   ↓
3. 自己資本比率の水準
   ↓
4. キャッシュフローの健全性
   ↓
5. 3期トレンドの確認
```

**重要ポイント**:
- 利益（P/L）と現金（CF）は別物
- 赤字は1期なら許容可能、2期以上は困難
- 新リース会計基準で見かけの負債が増える
