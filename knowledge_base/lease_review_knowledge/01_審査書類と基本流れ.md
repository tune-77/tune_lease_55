# リース審査：必要書類と基本流れ

**作成日**: 2026-05-30  
**情報源**: ネット検索  
**対象**: リース審査プロセスの標準化

---

## 📋 リース審査で必要な書類

### 必須書類（必ず提出）

#### 1. 決算書（財務諸表）
- **貸借対照表**: 企業の財務状況を長期的に示す
  - 資本金、総資産、負債額を確認
  - 自己資本比率を計算
  - 累積赤字の有無を確認
  
- **損益計算書**: その期の利益・損失を示す
  - 売上高（リース料支払い能力）
  - 経常利益（純粋な企業体質）
  - 営業CF（実際のキャッシュフロー）

- **キャッシュフロー計算書**
  - 営業CF、投資CF、財務CF
  - 実際の現金の流れを確認

#### 2. 代表者関連書類
- 本人確認書類
- 代表者の年齢・信用情報

### 条件付き提出書類

#### 3. 業種による追加書類
| 業種 | 必要書類 |
|------|---------|
| 建設業 | 営業許可証 |
| 製造業 | 営業許可証・工場登録書 |
| 飲食業 | 営業許可証・食品衛生責任者証 |
| 医療業 | 医師免許・医療機関指定書 |
| 士業 | 資格証明書 |

#### 4. その他提出書類
- 滞納履歴の有無確認
- 既存リース契約の支払い状況
- 銀行口座の定期通帳（6ヶ月以上）

---

## 🔍 審査で見られる5つの項目

### 1. **業績** ⭐⭐⭐ （重要度: 最高）

```
売上高 → リース料支払い能力を判断
経常利益 → 企業の純粋な体質
営業CF → 実際のキャッシュ状況
```

**判定基準**:
- ✅ 2期以上連続黒字: 審査通過可能性 高
- ⚠️ 直近1期赤字: 慎重に審査
- ❌ 2期以上連続赤字: 審査困難

### 2. **業種** ⭐⭐⭐ 

```
信頼度の高い業種          信頼度の低い業種
↓                         ↓
医師、弁護士など士業      美容・エステ
公務員                   風俗業
大企業の関連会社         飲食業（個人経営）
```

**背景**:
- 士業: 国家資格が必要 → 社会的信頼性高
- 飲食業: 起業しやすい反面、短期閉店も多い → 審査厳しい

### 3. **滞納履歴** ⭐⭐⭐

```
公共料金、光熱費         → 直近1年未払いなし
既存リース・ローン        → 支払い遅延なし
税金（法人税等）         → 滞納なし
```

### 4. **代表者の年齢・属性** ⭐⭐

- 若すぎる: 経験不足を懸念
- 適齢期（40-60代）: 審査通りやすい
- 高齢（65歳以上）: 事業継続性を懸念

### 5. **代表者の信用情報** ⭐⭐

- 個人ローン・クレジット: 支払い遅延なし
- 個人破産履歴なし
- 他企業での代表者経験（良好な前歴）

---

## 📊 創業年数による判定の違い

```
設立3年未満:   決算書が重視される
               → 1期目、2期目の決算がすべて
               
設立3年以上:   経営の安定性を見る
               → 過去3年の推移を重視
               
設立10年以上:  企業体質が確立
               → 信頼度が大幅に向上
```

## 🚨 審査が通りにくい場合

### ❌ 審査不可能なケース

1. **赤字決算**
   - 連続赤字（2期以上）
   - 累積赤字で債務超過

2. **滞納履歴**
   - 税金滞納
   - 既存リース・ローン延滞
   - 公共料金未払い

3. **信用情報の問題**
   - 個人破産経歴
   - 連続クレジット滞納

4. **業種の制限**
   - 風俗業
   - 一部の飲食業
   - 実績3ヶ月未満

### ⚠️ 審査が厳しくなるケース

- 新規事業（赤字傾向）
- 業績の急激な低下
- 代表者の高齢化
- 初めてのリース契約
- 小規模企業（売上 < 1000万円）

## ✅ 審査を通すためのポイント

### 1. 決算書の整備
```
→ 帳簿の正確性を確保
→ 脱税の疑いをなくす
→ 実績の信頼性を高める
```

### 2. 滞納なし
```
→ リース料の支払い実績
→ 税金・公共料金の支払い確認
→ 個人のクレジット履歴改善
```

### 3. 提出書類の完全性
```
→ 決算書3年分を揃える
→ 代表者の身分証提出
→ 業種に応じた許可証提出
```

### 4. 事業の安定性をアピール
```
→ 既存顧客との長期契約実績
→ 業界での信頼構築
→ 継続的な売上推移
```

## 📌 まとめ

リース審査は以下の優先順位：

```
1. 業績（売上・利益・CF）    ← 最重要
2. 業種・滞納履歴
3. 代表者の属性・信用
4. 提出書類の完全性         ← 形式要件
```

**成功の鍵**: 決算書の信頼性と支払い履歴の実績
